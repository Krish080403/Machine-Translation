सोबेन लेकाते भ्रष्ठाचार सोंगड़ा होड़ोकोआः व्यवस्था नातेन बिसुआस राकाब तादा।
इनिआः आपान योग्यता, पेड़ेः नातेन एटाः कोआः ओकोआ खींसोः ताना एना दिसुम रआः उनुबार रे नोकसान रिकाएआ।

"""आ ́ इतुआना, नेआ रेआः गोनोङ आञाः ओम लागातिङा, आ ́ाः माइन ओम लागातिङा, मेनदो दिसुम ते माराङ आ ́ाः माइन का होबा दाड़िओःआ, दिसुम रआः कुमु कोते माराङ आ ́ाः कुमु का होबाओःआ।"""
ओड़ोः एना मेनते ईमानदारी लोःते भ्रष् टाचार लोः आञाः लाड़ाइ सेसेनआ, जोरते सेसेनआ ओड़ोः भ्रष् टाचार कोरे कमि जारुड़ु होबाओआ।
आञ भ्रष्टाचार को नातेन बोरो रआः वातावरण बाइ सानाङ ताना चिःचि दिसुम रेन सोंगड़ा होड़ोको रे रेपेः रआः ओकोआ पारोम ताना एना पारोम आञाः होबाओआ।
"""मेनदो सोबेनाते माराङ नावा चुनौती हिजाकाना, भ्रष्टाचार कोलोः लेल बापइ दो मेनाः गेआ, मेनदो समाज जिदन रे चेतान स्तर रे मिआद ओको बदलि हिजाकाना, एनाको सोबेनाते माराङ चुनौतीओ मेनाः ओड़ोः मिआद समाज नातेन सोबेनाते माराङ उड़ुओ मेनाः।"""
चानाः ओकोए उड़ुः दाड़िआः चि आञाः दिसुम रे नामिनाङ मापाराङ संविधान आबुताः ताइकेन रेओ हुड़िङ लेका नेकान होड़ोको उड़ुङोः ताना ओकोनको भ्रष्टाचार राः मानातिङ ताना।
खुलेआम भ्रष्ठाचार राः जय-जयकार तानाएः।
समाज रे नेकन जङ रोआ रआः ओकोआ कोसिसि होबाओः ताना। भ्रष्ठाचार रआः ओकोआ महिमामंडन होबाओः ताना। भ्रष्ठाचार कोआः स्वतंत्रता हाराओ रआः ओकोआ निरंतर कोसिसि सेसेन ताना एना सोबेन समाज नातेन इसु माराङ चुनौती बाइआकाना। इसु माराङ उड़ुः रआः विषय बाइआकाना। भ्रष्टाचार कोलोः सानामाङ, समाज रे साङगिनते बाइआकान जोगाव रआः गे जान भ्रष्टाचार होड़ो एन होरा सेनोः रेआः को बोरोआ।
"""जदि आएआः महिमामंडन होबाओः रेदो, ओकोए तिसिङ भ्रष्ठाचार काएः रिकाएआ, आएआःओ नेकादो समाज रे माइन रआः रंग बाइओगा, एन होरा रे सेनोः का बानोः।"""
आञाः दुलाड़ा दिसुम रेनको।
बांग्लादेश रे ओकोआ गे होबाकाना एना मेनते होपोर जपाः दिसुम लेकाते उड़ुः मेनाः मेनेआः आञ बुजाओ दाड़िआः।
आ ́ आशा ताना चि एनताः रे दसा सेकेड़ागे सोंगड़ाओःआ। पुराःते 140 करोड़ दिसुम रेन होड़ोकोआः उड़ुः चि एनताः हिन्दु को, एनताः रेन होड़ोको जाआं लेकाते एन समाज रेआः बनचाओ जाआं लेका मेनाः।
भारत सोबेन इमतङ सनङ ताना चि आबुआः होपोर जपः रेन दिसुम सुकु ओड़ोः सुकु रआः होरा तेबु सेनेका।

"""आबु हिनिजुः सोमाए रे बांग्लादेश रआः उनुबार होरा रे सोबेन सोमाए आबुआः बुगिन उड़ुः गे ताइना, चिआःचि आबु होड़ो जाति रआः बुगिन उड़ुः होड़ो को मेनाःबुआ।"""
आञाः दुलाड़ा दिसुम रेनको।
आबुआः संविधान के 75 साल होबाओःताना।
"""भारत रआः संविधान रआः 75 साल सेसेन, दिसुम रआः मिआद बाइ, दिसुम चेतान बाइ रे इसु माराङ कामि ताना।"""
भारत रआः लोकतंत्र के पोकोतो ओम मेनते आबुआः दिसुम रआः संविधान रआः इसु पुराः गितिः ताना।
आबुआः दिसुम रेन दलित, पीड़ित, शोषित, वंचित को बनचव रआः इसु माराङ कामि आबुआः संविधान रआः कामि रिकाकादा।
नाःदो आबु संविधान रआः 75 साल मानातिङ सेनोः ताना तोबे आबु दिसुम रेनको संविधान रे तोलाकान कामि रआः भाव रे पेड़ेः ओम पुराः जारूड़ु मेनाः आर ओको इमताङ आञ कामि रआःञ जागार ताना तोबे आञ होड़ोको रे बोझ बाइ काञ सानाङा।
कामि केन्द्र सरकाराःओ मेनाः। कामि केन्द्र सरकाराः सोबेन होनकोआःओ मेनाः। कामि राइज सरकाराःओ मेनाः। राइज सरकाराः होनकोआः मेनाः।
कामि सोबेन हातु रआः स्वाराज संस्था रआः मेनाः। सानाङ दो पंचाइटि होबाओः, नगरपालिका होबाओः, महानगरपालिका होबाओः, तहसिल होबाओः, जिला होबाओः सोबेन कोआः कामि मेनाः।
मेनदो एनलोःते 140 करोड़ दिसुम रेनकोआः कामि मेनाः।
"""जुदि अबु सोबेन जमा कोअते अपनः कमि को बु समबड़व रेदो अबु अपनः ते एटअः कोआ अधिकार को बनचव मेनते कमि कोबु बइया ओड़ोः, ओकोन सोमय कमि मनातिङ होबाओ तना, एन सोमय अधिकार को बनचव कन तइना, एना मेनते जांए एटअः कमि रेअः जारूड़ु का होबाओ तना।"""
आञ सानाङ ताना चि आबुआः ने भाव लोःते बु सेनेआ। आबुआः लोकतंत्रो पोकोतोआ।
आबुआः पेड़ेः ओड़ोः हाराओआ आर मिआद नावा पेड़ेः लोःते आयारतेबु सेनेआ।
आञाः दुलाड़ा दिसुम रेनको।
आबुआः दिसुम रे Supreme Court रे खनेसा मिसा Uniform Civil Code नातेन राः जपागर होबाकाना।
चिमिन सा हुकुम को ओमाकादा चिआः चि दिसुम रे मिआद इसु पुराः को मनतिङ तना ओड़ोः नेआ रे सारतेओ मेनाः चि ओकोआ Civil Code आबु आउकेआते जिदेन तना एना Civil Code सारते दो मिद किसिम रआः Communal Civil Code तनाः, भेदभाव रआः Civil Code तनाः।
नेकान Civil Code आते चिमताङ संविधान रआः 75 साल मानातिङ ताना आर संविधान रआः भावो ओकोआ आबु काजि ताना मेनते दिसुम रआः Supreme Courtओ आबु काजि ताना मेनते ओड़ोः एन सोमय ओकोआ संविधान बाइतानकोआः कुमु ताइकेना एन कुमु पुराओ आबुआः कामि रेओ आ'र आ ́ मानातिङ ताना चि नेआ गंभीर विषय रे दिसुम रे जपागर होबाओः आ, ओड़ोः बेसे जपागर होबाओआ।
"""सोबेन अकोअः उडुः को आउ केआते हिजुः लेना ओड़ोः अइन को, ओकोआ अइन धोरोम लेकाते दिसुम को हटिङ जदा, ओकोआ चेतान लातार रआः सुसि को बइओगा, नेकन अइन को नावा समाज रे जेतान ठांव का दड़िओआ ओड़ोः एना मेनते अइङ दो काजिआ, नाः दो सोमाए रआः अनासि, नाः दो दिसुम रे मिआद Secular Civil Code होबाओआ, आबु Communal Civil Code रे 75 साल बु ओमजदा।"""
नाःदो आबु Secular Civil Code साः सेनोः लागातिङा आर एनताः सेनकेआते दिसुम रे धरम लेकाते ओकोआ भेदभाव होबाओः ताना। सोंगड़ा होड़ोको सांगिन टोओरोः ताना एना रे आबु राटि नामोगा।

तिसिङ आबु लेल जादा आञाः सानामाङ रे ओकोए दाङगाड़ाको मेनाः कोआ एना रे ओलाकाना। माइ बharat ओकोआ संगठन रआः लुतुम मेनाः एना रेआः काजिञ ओलाकादा।
